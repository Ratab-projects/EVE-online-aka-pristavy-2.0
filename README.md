# EVE online / Stellaris aka Přístavy 2.0
Hra založená na podobné myšlence jako universum. Prvky ze hry přístavy. 
Každý tým má svou kosmickou loď, kterou si v průběhu hry upravuje a dovybavuje. Hráči se mohou profilovat v různých herních činnostech, jako jsou těžba, obchod, převoz surovin, boj.
